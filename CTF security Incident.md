In this lab, we will investigate a CTF security incident involving a compromised web server.

- The lab consists of one machine running a vulnerable web service that has been accessed by an attacker.

Throughout this lab, :

Analyze Apache web server logs to identify suspicious activity

Investigate network capture files to uncover leaked credentials

Assess credential strength and perform basic hash cracking techniques

Identify unauthorized access via system and authentication logs

Detect and remove persistence mechanisms used by an attacker

Terminate malicious processes and secure the system
